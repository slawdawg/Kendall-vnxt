# KNX Source Packet Examples

Date: 2026-06-01

Status: local metadata-only examples

## Scope

These examples cover the three approved first source packet classes:

- `user-authored-planning-document`
- `public-or-synthetic-sample-data`
- `generated-report`

The examples are metadata-only. They do not copy source contents, do not mutate source, and do not approve external sends or operational source intake.

## Example Packet Set

Machine-readable examples:

- `source-packet-examples-2026-06-01.json`

## Boundary Notes

Excluded from these examples:

- runtime evidence inventory as a source packet class,
- exported files or attachments,
- customer/project data,
- production systems,
- credentials, tokens, MFA, account/security material,
- GitHub/remotes,
- external providers,
- local model/GPU-derived outputs,
- source mutation records,
- operational source intake.
